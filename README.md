# Google Trends API Crawler

This project demonstrates crawling Google Trends using Python and deploying on Google Cloud Functions with Pub/Sub & Scheduler.

## Features
- Crawl trends based on keywords
- Schedule recurring jobs on Google Cloud
- Save results to Google BigQuery

## Skills Highlight
- Python, API usage
- Cloud deployment (GCP Functions, Pub/Sub, Scheduler)
- Logging, scheduling, automation
